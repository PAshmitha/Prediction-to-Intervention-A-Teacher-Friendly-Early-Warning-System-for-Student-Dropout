# Prediction-to-Intervention-A-Teacher-Friendly-Early-Warning-System-for-Student-Dropout
This project is a Machine Learning-based Early Warning System designed to predict student dropout risk and help educators take timely action.  It analyzes academic, demographic, and socio-economic data to identify students who are at risk of dropping out and provides interpretable results with actionable insights.
## 🎯 Objective
* Predict student dropout at an early stage
* Provide teacher-friendly insights
* Enable timely interventions to improve student retention
## 🚀 Features
* 📊 Predicts dropout risk using ML models
* 📈 Uses multiple algorithms (LR, DT, RF, XGBoost)
* 🎯 Risk classification:

  * High Risk
  * Medium Risk
  * Low Risk
* 📉 Feature importance for explainability
* 🖥️ Dashboard for easy analysis
* 📂 Supports CSV upload & manual input
## 🧠 Technologies Used
* Python
* Machine Learning
* XGBoost
* Scikit-learn
* Pandas, NumPy
* Data Visualization
## ⚙️ Methodology
1. Data Collection (academic, demographic, socio-economic)
2. Data Preprocessing (cleaning, encoding, scaling)
3. Feature Selection
4. Model Training:

   * Logistic Regression
   * Decision Tree
   * Random Forest
   * XGBoost
5. Model Evaluation (Accuracy, Precision, Recall, F1-score)
6. Risk Classification System
7. Dashboard Visualization
## 📊 Model Performance
* XGBoost chosen as best model
* Balanced performance:

  * Accuracy: ~0.82
  * Recall: ~0.44
  * F1-score: ~0.40
## 📉 Risk Classification
* 🔴 High Risk → Immediate intervention
* 🟡 Medium Risk → Monitoring required
* 🟢 Low Risk → Low probability of dropout
## 📷 System Output
* Student risk prediction
* Feature importance visualization
* Dashboard insights for educators
## 💡 Key Advantages
* Simple and interpretable model
* Real-time insights for teachers
* Easy to use (non-technical users)
* Helps in improving student retention
